# Projektideen Sammlung
Auf unseren [Discord Server](https://discord.gg/VjrfCFKRgR) von [BlackScorp](https://www.youtube.com/VitalijMik) wird immer mal wieder nach Ideen für Projekte gefragt.
Hier wollen wir für euch eine kleine Sammlung mit Ideen aufbauen.

Gerne könnt ihr selber Ideen per Pull Request einreichen.
